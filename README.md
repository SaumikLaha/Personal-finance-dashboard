# 💰 Personal Finance Dashboard


 ## Live Demo: https://personal-finance-dashboard-nu.vercel.app/



---

## 📌 What Is This?

Personal Finance Dashboard is a modern web application built with React and Tailwind CSS that helps users track their income, expenses, and financial insights in a clean and interactive UI.

It provides role-based access, analytics, and a smooth user experience similar to real-world financial tools.

---

## 🧠 Core Concepts

- React Component Architecture
- Context API for State Management
- Role-Based Access Control (Admin / Viewer)
- Data Visualization using Charts
- Clean UI/UX with Tailwind CSS
- Local Storage for Persistence

---

## ✨ Key Features

- 🔐 Authentication (Admin & Viewer)
- 📊 Dashboard with Summary Cards
- 📈 Monthly Balance Trend (Line Chart)
- 🥧 Expense Category Distribution (Pie Chart)
- 📋 Transactions Management
    - Add Transaction (Admin only)
    - Edit Transaction
    - Delete Transaction
- 🔍 Search & Filter Transactions
- 🌙 Dark Mode Support
- 💡 Smart Insights (Auto Analysis)
- 📭 Empty State Handling

---

## 🌐 Platform-Wide Features

- Responsive Design (Mobile + Desktop)
- Smooth Hover & Transitions
- Clean Layout Structure
- Reusable Components
- Scalable Code Architecture

---

## 🛠 Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- 📊 Recharts
- 🌐 Vercel (Deployment)
- 📦 Context API (State Management)

## ⚙️ End-to-End Setup

### 1. Clone Repository

```bash
git clone https://github.com/SaumikLaha/Personal-finance-dashboard.git
cd Personal-finance-dashboard
```bash

2. Install Dependencies
npm install
3. Start Development Server
npm run dev
4. Open in Browser
http://localhost:5173

```
## 📋 Prerequisites
```
Node.js (v16 or above)
npm or yarn
Basic knowledge of React
```
